Amusons nous avec javascript

Ce site vous permet de convertir un nombre de la base 2, 10 ou 16, dans les 2 autres bases restantes.
Il vous suffit de rentrer une valeur dans la case de votre choix et d'appuyer sur le bouton convertir.
Pour refaire une conversion, appuyer sur le bouton supprimer.

Cet outil ne fonctionne qu'avec toutes les cases vides sauf celle à traduire.
